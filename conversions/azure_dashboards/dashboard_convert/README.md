# Azure Dashboard convert

Converts a full Azure Portal dashboard export so `azurerm_portal_dashboard.dashboard_properties` accepts it. List/tuple `lenses` and `parts` become objects keyed by index strings (`"0"`, `"1"`, …); all other fields are unchanged.

## Inputs

| Name | Description | Type | Default | Required |
|------|-------------|------|---------|:--------:|
| <a name="input_dashboard_json"></a> [dashboard\_json](#input\_dashboard\_json) | Full Azure Portal dashboard export JSON (ARM envelope with a properties block). | `string` | n/a | yes |
## Outputs

| Name | Description |
|------|-------------|
| <a name="output_dashboard_properties"></a> [dashboard\_properties](#output\_dashboard\_properties) | Dashboard body JSON for azurerm\_portal\_dashboard.dashboard\_properties. |
